# Hyperspectral Image Recognition

高光谱图像识别毕业设计项目

## 项目结构

本项目包含以下主要模块：

### 目录结构

- model/ - 深度学习模型模块
- rag/ - RAG (Retrieval-Augmented Generation) 模块  
- server/ - 后端服务器模块
- web/ - 前端 Web 界面模块

## 分支说明

本项目采用多分支管理：

- **master**: 主分支，包含项目整体结构
- **branch-model**: 模型模块代码
- **branch-rag**: RAG 模块代码
- **branch-server**: 服务器模块代码
- **branch-web**: 前端 Web 模块代码

## 快速开始

### 克隆项目

```bash
git clone https://github.com/xiaoguo1224/Hyperspectral-Image-Recognition.git
```

### 切换分支获取特定模块

```bash
# 获取模型模块
git checkout branch-model

# 获取前端模块
git checkout branch-web

# 获取后端服务
git checkout branch-server

# 获取 RAG 模块
git checkout branch-rag
```

## 技术栈

### 后端
- Python - 主要编程语言
- 深度学习框架 - 用于模型训练和推理
- RAG - 检索增强生成

### 前端
- Vue.js - 前端框架
- Vite - 构建工具
- Pinia - 状态管理

## 项目功能

- 高光谱图像分类识别
- 深度学习模型训练与推理
- RAG 智能问答
- Web 可视化界面
- RESTful API 服务
